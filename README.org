*目标*
build qygame server compiler
包含gcc编译器 && 编译所需要的库

以docker image的形式交付. image托管在ghcr.io/qygame/qy-compiler


*构造流程*
1. trigger when push tag
2. main.yml call docker (Dockerfile安装配置第三方库)
4. main.yml push docker-image to ghcr.io/qygame/qy-compiler
