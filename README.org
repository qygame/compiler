*目标*
build qygame compiler
包含一个gcc, gdb 以及运行需要的dll(.so or .a)环境

以docker image的形式交付. image托管在[[https://hub.docker.com/repository/docker/clay2019/qy_compiler/general][dockerhub clay2019/qy_compiler]]


*构造流程*
1. trigger when push tag
2. main.yml call docker (Dockerfile安装配置第三方库)
4. main.yml push docker-image to dockerhub/clay2019/qy_compiler
